Some "hamburger menu" concepts without javscript

  • Concept 1

    Not really a conventional hamburger menu. It's simply a link to the bottom of the page where the menu has been placed.

  • Concept 2

    Uses focused-within css selector to move a full screen div from off screen to on screen when it, or any of its contents have been focused. Not supported by IE and older Edge versions.

  • Concept 3

    Uses target css selector to turn display from none to visible (in this case, flex) when targeted. Close button is empty anchor that is clickable. Works with most browsers, but fills browser history with anchor tags.
